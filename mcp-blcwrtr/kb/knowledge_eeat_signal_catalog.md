# E-E-A-T Signal Catalog

## Översikt

E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness) är Googles ramverk för att bedöma innehållskvalitet. Denna katalog definierar konkreta signaler som stärker E-E-A-T i backlink-content.

## Experience (Erfarenhet)

### Primära signaler
- **Personliga anekdoter**: "Under mina 10 år som..."
- **Specifika exempel**: Detaljerade case studies
- **Praktiska tips**: Steg-för-steg från verklig användning
- **Problemlösning**: "Jag stötte på X och löste det med Y"

### Implementering
```markdown
"Efter att ha analyserat över 500 släktträd har jag lärt mig att den vanligaste 
fallgropen är att anta stavningar. I mitt eget släktträd fanns 'Andersson' 
stavat på sju olika sätt mellan 1750-1850."
```

### Att undvika
- Generiska påståenden utan detaljer
- "Man säger att..." utan källa
- Teoretiska resonemang utan praktisk förankring

## Expertise (Expertis)

### Primära signaler
- **Facktermer**: Korrekt användning av branschterminologi
- **Djupanalys**: Går bortom ytlig information
- **Metodkännedom**: Beskriver etablerade processer
- **Källhänvisning**: Refererar till primärkällor

### Implementering per bransch

#### Genealogi
```markdown
"Enligt Riksarkivets riktlinjer för källkritik (RA-FS 2019:2) bör kyrkoböcker 
alltid korskontrolleras mot husförhörslängder för perioden 1686-1894."
```

#### Digital marknadsföring
```markdown
"Core Web Vitals, specifikt LCP (Largest Contentful Paint) under 2.5 sekunder, 
har sedan maj 2021 varit en bekräftad rankingfaktor enligt Google."
```

#### Finans
```markdown
"Enligt Basel III-regelverket måste banker upprätthålla en kärnprimärkapitalrelation 
om minst 4.5%, vilket påverkar utlåningskapaciteten."
```

### Expertismarkörer
- Branschcertifieringar omnämns
- Konferensdeltagande refereras
- Publikationer citeras
- Professionell bakgrund antyds

## Authoritativeness (Auktoritet)

### Primära signaler
- **Erkännande**: "Utnämnd till årets..."
- **Citat av andra**: "Som X påpekar i sin forskning..."
- **Unika insikter**: Original research eller data
- **Thought leadership**: Nya perspektiv på etablerade sanningar

### Auktoritetsbyggande element

#### Datadriven auktoritet
```markdown
"Vår analys av 10,000 svenska företagswebbplatser visar att endast 23% 
uppfyller WCAG 2.1 AA-standarden för tillgänglighet."
```

#### Positionsbaserad auktoritet
```markdown
"Som ansvarig för digitalisering på Lantmäteriet ser jag dagligen hur..."
```

#### Forskningsbaserad auktoritet
```markdown
"I den peer-reviewade studien publicerad i Journal of Digital Heritage (2023) 
demonstrerade vi att..."
```

### Auktoritetshierarki
1. **Myndighet/Institution**: Högst trovärdighet
2. **Akademisk**: Forskning och utbildning
3. **Branschledare**: Erkända företag
4. **Praktiker**: Yrkesverksamma med bevisad erfarenhet
5. **Entusiast**: Hobbyexpert med djup kunskap

## Trustworthiness (Trovärdighet)

### Primära signaler
- **Transparens**: Öppen om begränsningar
- **Balans**: Presenterar flera perspektiv
- **Faktakontroll**: Verifierbara påståenden
- **Uppdateringar**: Datum och versionshantering

### Trust-byggande komponenter

#### Källtransparens
```markdown
"Statistiken kommer från SCB:s databas (hämtad 2024-01-15, tabell BE0101A). 
Observera att siffrorna är preliminära och kan revideras."
```

#### Intressekonflikt-deklaration
```markdown
"För transparensens skull: Jag har tidigare arbetat som konsult åt tre av 
de nämnda företagen, dock inte de senaste två åren."
```

#### Balanserad framställning
```markdown
"Medan metod A ger snabbare resultat (2-3 dagar), ger metod B mer tillförlitliga 
data (95% träffsäkerhet vs 78%). Valet beror på dina prioriteringar."
```

### Trust-sänkande element att undvika
- Överdrifter: "Alltid", "Aldrig", "Garanterat"
- Dolda affiliates: Ej deklarerade kopplingar
- Utdaterad info: Gamla datum utan uppdatering
- Ensidighet: Endast positiva aspekter

## Signalkombinationer

### Stark E-E-A-T profil
```markdown
[Experience] "Under mina 15 år som legitimerad finansiell rådgivare har jag..."
[Expertise] "...tillämpat Modern Portfolio Theory enligt Markowitz principer..."
[Authority] "...vilket resulterade i att Finansinspektionen citerade vårt ramverk..."
[Trust] "...Fullständig metodbeskrivning finns i bilaga A, peer-reviewad av..."
```

### Medium E-E-A-T profil
```markdown
[Experience] "Jag har testat fem olika genealogi-tjänster..."
[Expertise] "...och jämfört deras källdatabaser mot Riksarkivets standard..."
[Authority] "...Resultaten presenterades på Släktforskardagarna 2023..."
[Trust] "...Rådata finns tillgänglig på GitHub för egen verifiering."
```

## Branschspecifika E-E-A-T

### YMYL (Your Money or Your Life)
Kräver högsta nivå av alla signaler:
- Medicinsk info: Läkarlegitimation + källhänvisning
- Finansiell rådgivning: Certifiering + disclaimer
- Juridisk vägledning: Juridiksexamen + aktualitet

### Hobby/Fritid
Kan klara sig med lägre formell expertis:
- Erfarenhet viktigast
- Community-erkännande väger tungt
- Praktiska resultat övertrumfar teori

### B2B/Professional
Balans mellan alla fyra:
- Branscherfarenhet critical
- Case studies avgörande
- ROI-data förväntas
- Professionell ton genomgående

## Mätning och validering

### E-E-A-T Scorecard
```
[ ] Experience: Minst 2 konkreta exempel
[ ] Expertise: 3+ facktermer korrekt använda  
[ ] Authority: 1+ extern referens/citering
[ ] Trust: Källhänvisning + balans + transparens
```

### Kvalitetsnivåer
- **Excellent** (4/4): Alla kategorier väl representerade
- **Good** (3/4): En kategori kan förbättras
- **Acceptable** (2/4): Minimikrav för publicering
- **Poor** (0-1/4): Kräver omfattande revision

## Best Practices

1. **Frontload E-E-A-T**: Etablera trovärdighet tidigt
2. **Visa, berätta inte**: Konkreta exempel > påståenden
3. **Lager på lager**: Bygg signaler genom hela texten
4. **Branschanpassning**: Matcha förväntad expertis nivå
5. **Uppdatera regelbundet**: Håll information aktuell

## Varningssignaler

### Red Flags
- Motstridiga påståenden
- Uppenbart felaktig information
- Överdriven självsäkerhet
- Avsaknad av nyanser
- Plagierade expertuttalanden

### Åtgärder
1. Faktakontroll alla påståenden
2. Lägg till källhänvisningar
3. Inkludera disclaimers
4. Be om expertgranskning
5. Överväg omskrivning

Denna katalog uppdateras löpande baserat på Googles riktlinjer och branschens best practices.
